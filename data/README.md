# Data for iUma22

In addition to the supplementary following data was used for the construction of iUma22.

| Item          | Content                                                      |
| ------------- | ------------------------------------------------------------ |
| `Article`     | All material directly relevant for the publication, e.g. manuscript files, supplementary, figures. |
| `BIOLOG`      | Results for the Biolog Phenotype arrays and the correlation with simulations. |
| `GrowthRates` | Excel sheets with single growth experiments as well as summary Excel with growth and uptake rates. |
| `MSc_Lieven`  | Fundamental work and Master thesis of Christian Lieven.      |
| `Sequencing`  | Details regarding the pan-genome of *Ustilago maydis* strains. |
