# Code for iUma22

The `code` subdirectory provides workflows to modify and analyze iUma22. 



| Item                              | Content                                                      |
| --------------------------------- | ------------------------------------------------------------ |
| `AnnotatePthwTlsModel`            | Folder with Matlab-functions to annotate the raw output from PathwayTools. |
| `Maps`                            | Folder with Metabolic maps to visualize flux distributions for Glycolysis+TCA and glycine pathways. |
| `iUma22_BiotechSimulations.ipynb` | 1. Simulations to reproduce growth reduction with increasing acetate concentration during glucose-acetate co-feeding. 2. Calculation of maximum itaconate production yield. |
| `iUma22_EscherMapping.ipynb`      | Example code for Escher visualization                        |
| `iUma22_FASTA+KAAS.ipynb`         | Workflow for fasta sequence analysis and KAAS analysis.      |
| iUma22_ModelCorrections.ipynb     | Manual model adaptations based on charge imbalance and Biolog arrays. |
